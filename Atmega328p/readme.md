This project contains a custom PCB layout for the ATmega328P microcontroller, along with supporting components such as voltage regulation, LEDs, connectors, and I/O ports. The design is created in Autodesk EAGLE and intended as a general-purpose microcontroller development board.

The board includes:

ATmega328P (DIP-28 package)

7805 voltage regulator

Input power connector

LEDs and resistors

External pin headers for I/O

Supporting discrete components and connectors
